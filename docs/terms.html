2. Description of Service

Sol Clock is an analogue clock and information dashboard application designed for the Solana Seeker phone (Android) that displays:

  • An analogue clock face with customisable display options
  • Live cryptocurrency price data for SOL (Solana) and SKR (Seeker) tokens
  • Solana blockchain slot height
  • World clock times with automatic daylight saving adjustments
  • Wallet balance display (when the user opts in via wallet sign-in)
  • Seeker Genesis Token (SGT) verification status
  • Home screen widget with live price data
  • Screensaver (DreamService) and nightstand modes

The App functions as a read-only information display tool. It does not and cannot execute financial transactions, swap tokens, transfer funds, manage portfolios, or provide financial advisory services of any kind.
